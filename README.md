# Система мониторинга

Разработка системы мониторинга, анализа информации и создание платформенного решения на основе ИИ для прогнозирования социально-экономических показателей предприятий и региональной власти

## Настройка рабочего окружения

**Приступая к работе:**

**Установка зависимостей и запуск проекта:**

Требуется **Node.js: 22.x (LTS)**[

```bash
# Установка зависимостей
$ npm install

# Запуск в режиме разработки на localhost:5173
$ npm run dev
```

## Стек

- **Сборщик:** [Vite](https://vite.dev/guide/) — `vite@7.3.1`
- **Роутинг:** [React Router DOM](https://reactrouter.com/en/main) — `^7.13.1`
- **Установщик зависимостей:** [npm](https://docs.npmjs.com/)
